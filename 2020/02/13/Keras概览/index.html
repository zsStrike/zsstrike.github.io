函数式 API

层叠式模型是一种将多个层之间连接的简单模型，我们可以使用 Keras 中的函数式 API 来构建复杂的模型：

  • 多个输入模型
  • 多个输出模型
  • 包含共享层（同一个层被多次调用）的模型
  • 不包含顺序流的模型（如残差模型）

Callbacks

callback 对象用于传给模型以此在训练期间被使用，可以使用自定义的 callback，同样可以使用内建的 callback：

  • tf.keras.callbacks.ModelCheckpoint: 每次迭代保存检验点
  • tf.keras.callbacks.LearningRateScheduler: 动态改变学习速率
  • tf.keras.callbacks.EarlyStopping: 如果模型没有改经就停止训练
  • tf.keras.callbacks.TensorBoard: 监视模型的行为
